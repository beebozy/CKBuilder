# CKBuilder
# CKB Learning Repository

A personal repository dedicated to learning the **Nervos CKB (Common Knowledge Base)** ecosystem from the ground up.

This repo documents my journey exploring:

* CKB scripts
* Cell model fundamentals
* Transactions and witnesses
* Lock scripts and type scripts
* CKB-VM concepts
* Molecule serialization
* On-chain verification logic
* Tooling and SDKs
* Smart contract experiments
* Testing and debugging workflows

---

## Goals

The purpose of this repository is to:

* Build a strong understanding of how CKB works internally
* Learn by writing and testing scripts manually
* Document concepts with examples and notes
* Experiment with different contract patterns
* Create reusable references for future projects

## Why This Repo Exists

CKB is very different from account-based blockchains, and the best way to learn it is by building things directly.

This repository serves as:

* a learning journal,
* a sandbox for experiments,
* and a long-term reference for understanding the Nervos ecosystem.
